# Security Policy

## Supported versions

HubRoute 当前处于 alpha 阶段，仅维护最新的 0.1.x 版本和 main 分支。

## Reporting a vulnerability

请使用 GitHub 仓库的 **Security > Report a vulnerability** 私下提交安全报告，不要创建公开 Issue。

报告应尽量包含：

- 受影响版本和操作系统
- 复现步骤或最小示例
- 可能泄露的数据或可执行的操作
- 建议的缓解方式（如有）

涉及代理凭据、安装包校验绕过、任意进程启动、路径覆盖或供应链风险的问题会优先处理。确认问题后，维护者会在修复发布前协调披露时间。
